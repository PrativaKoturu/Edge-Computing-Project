Start On-Cloud Simulation:
  $ cd edge-rl-oncloud/firmware
  $ code .
  → Press F1 → "Wokwi: Start Simulator"

Start On-Device Simulation:
  $ cd edge-rl-ondevice
  $ docker compose up -d                    # Start Docker broker
  $ cd firmware && code .
  → Press F1 → "Wokwi: Start Simulator"

Run Both Simultaneously:
  Terminal 1:  cd edge-rl-oncloud/firmware && code . [F1 → Wokwi Start]
  Terminal 2:  cd edge-rl-ondevice && docker compose up -d && cd firmware && code . [F1 → Wokwi Start]
  Terminal 3:  streamlit run dashboard/app.py        [Monitor at localhost:8501]
